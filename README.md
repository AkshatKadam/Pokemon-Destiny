# Pokémon Destiny

A fan-made Pokémon game built on **[Pokémon Essentials](https://essentialsdocs.fandom.com/) v21.1** / RPG Maker XP.
Set in the original region of **Volua** — a Norse/Germanic-inspired world of three kingdoms, fallen monarchies,
and a militarist power called **Team Might**. You play a person who has lost more than they remember, drawn back
into the world by a small Pokémon and the people who need them.

> ⚠️ **Work in progress.** This is an in-development fangame, not a finished release. Non-commercial fan project;
> Pokémon is © Nintendo / Game Freak / The Pokémon Company.

## Status

Currently building **the foundations and a polished Prologue** as the first shippable slice — the opening on
Eyjaf island through to the start of the journey at Ibitha City. Later acts are designed but not yet built.

- ✅ Engine on **Essentials v21.1**
- ✅ Core early-game maps built (Dreidorf, Reise, Ibitha and surrounds)
- ✅ Curated regional wild encounters for the starting island
- ✅ **Following Pokémon** (HGSS-style) integrated
- 🔨 Prologue story events — in progress
- ⏳ Custom region map, title screen, and original art — planned

## Region: Volua (and neighbours Sylia, Sifrier, Torria)

Volua is built on a Norse mythos — sacred trees, three realms, and a peace that was never quite real. The world,
map, characters, and story are original.

## Features (planned & in progress)

- Original story, world, and cast
- HGSS-style following Pokémon
- A diegetic take on HMs and gym badges woven into the world's politics
- Gen 4/5-style presentation

## Running the project

This repo is the RPG Maker XP project. To run or edit it you need **RPG Maker XP** and the **Pokémon Essentials
v21.1** base it's built on.

- **Play (debug/test):** open `Game.rxproj` in RPG Maker XP and press **F12**.
- **Audio** is not included in the repo (large files); the game still runs, some tracks will be silent.

## Repository layout

```
Data/        RPG Maker + Essentials data (maps, scripts, compiled data)
PBS/         Essentials text data (species, moves, encounters, map metadata, …)
Graphics/    Tilesets, characters, sprites, UI
Plugins/     Essentials plugins (Following Pokémon EX, v21.1 Hotfixes, dev tools)
Fonts/       Fonts
Game.rxproj  RPG Maker XP project file
```

Design documents (`Design/`) are kept local; a few reference docs are tracked here:
`Design/Prologue-Spec.md`, `Design/Prologue-Roster-and-Encounters.md`, `Design/Pokemon-Destiny-HANDOFF.md`.
> Note: `Pokemon-Destiny-HANDOFF.md` contains **full-game design spoilers**.
